本次更新补齐了桌面端设置能力，重点完善主题、主色和应用信息展示，并修复设置弹窗在主题切换过程中的稳定性问题。

## 新增内容

- 新增设置弹窗，采用左侧菜单、右侧内容的双栏布局。
- 新增外观设置页，支持在 `亮色`、`暗色`、`跟随系统` 三种主题模式之间切换。
- 新增主色切换能力，支持多种常见主色方案，并会持久化保存。
- 新增关于页，展示应用图标、应用名称、版本号，并提供手动“检查更新”按钮。

## 体验改进

- 设置弹窗改为固定高度，滚动区域只保留在右侧内容区，布局更稳定。
- 左侧菜单激活态去除阴影，视觉层级更简洁。
- 主色方案会同步作用于侧栏与主要交互控件，整体主题更统一。

## 问题修复

- 修复设置弹窗在切换主题、尤其是切换“跟随系统”时会意外关闭的问题。
- 修复设置弹窗关闭行为异常的问题，恢复了遮罩关闭、关闭按钮关闭和 `Esc` 关闭。
- 修复主题与主色偏好未完整持久化的问题，应用重启后可以恢复上次设置。
